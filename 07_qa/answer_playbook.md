# 作答手册

## 总原则

- 不停留在名词解释，至少补上机制、条件、组织变化和性能后果。
- 能跨章节就跨章节，优先把问题挂到“热力学-相图-相变”或“缺陷-界面-扩散”等主线。
- 遇到比较题，固定从驱动力、扩散特征、组织形貌、性能影响四个维度回答。
- 遇到工艺题，固定按“工艺条件变化 → 组织演化 → 性能变化 → 应用或局限”作答。

## 机制解释题

- 使用场景：用户问为什么会发生、驱动力是什么、过程如何演化。
- 推荐步骤：先定义研究对象 → 说明驱动力 → 写出必要条件 → 分阶段描述过程 → 交代组织变化 → 落到性能后果

## 比较题

- 使用场景：用户要求比较两种结构、两类相变或两种机制。
- 推荐步骤：分别定义对象 → 比较驱动力或扩散特征 → 比较组织与形貌 → 比较性能后果 → 总结最易记的差异点

## 相图/曲线分析题

- 使用场景：用户要求读图、走路径、解释组织演化。
- 推荐步骤：先认坐标和相区 → 标关键点和关键线 → 沿成分或温度路径分析 → 翻译成组织变化 → 必要时补杠杆定律或自由能解释

## 工艺-组织-性能题

- 使用场景：用户问某工艺如何改变材料行为。
- 推荐步骤：先说工艺改变了什么条件 → 再说组织如何响应 → 再说性能如何变化 → 最后补适用场景与局限
