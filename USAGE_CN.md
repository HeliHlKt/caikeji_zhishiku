# 材料科学基础知识库使用说明

这份说明适合发给获得仓库访问权限的使用者，帮助他们从 GitHub 下载并正确使用这个知识库。

## 1. 获取仓库

本仓库是私有仓库，使用前需要：

- 登录 GitHub
- 获得该仓库的访问权限

拿到权限后，可以用以下两种方式获取仓库内容。

### 方式一：使用 Git 克隆

```powershell
git clone https://github.com/HeliHlKt/caikeji_zhishiku.git
```

更新仓库时，在本地目录执行：

```powershell
git pull
```

### 方式二：在 GitHub 页面下载 ZIP

在仓库页面点击：

- `Code`
- `Download ZIP`

下载后解压即可使用。

## 2. 仓库主要内容

- `00_source/`
  原始 PDF 文件

- `02_ocr_pages/`
  按页 OCR 文本，适合精确回溯页面内容

- `03_chunks/`
  切分后的知识片段，适合检索与问答引用

- `04_index/`
  章节、页码、主题、切片等索引文件

- `05_cards/`
  概念卡片与知识点整理

- `06_topics/`
  主题导读与章节说明

- `07_qa/`
  面向问答的辅助框架

- `08_outputs/`
  已整理好的摘要、汇总与可直接使用的结果

## 3. 推荐使用顺序

第一次使用时，建议按下面顺序查看：

1. `README.md`
2. `08_outputs/ready_for_qa.md`
3. `04_index/chapter_map.json`
4. `04_index/topic_routes.json`
5. `07_qa/answer_playbook.md`

如果只是想快速了解内容，优先看：

- `08_outputs/`

如果要精确查找某个知识点，优先看：

- `04_index/`
- `03_chunks/`
- `02_ocr_pages/`

## 4. 如何查找内容

常见检索方式有三种：

### 按章节找

先看：

- `04_index/chapter_map.json`

适合确认某个概念大致位于哪一章、哪一节。

### 按主题找

先看：

- `04_index/topic_routes.json`
- `06_topics/chapter_guides.md`

适合查找某个机制、概念、工艺或材料行为。

### 按原文回溯

先看：

- `03_chunks/`
- `02_ocr_pages/`

适合需要结合教材原始表述、上下文和页码定位时使用。

## 5. 如何配合 AI 使用

如果把这个仓库提供给 AI 或问答助手使用，建议在提问时明确说明要基于这个知识库回答。

推荐提问方式：

- 请使用材料科学基础知识库，解释位错强化的基本机制
- 请使用材料科学基础知识库，找出“变质处理”出现在哪一章
- 请使用材料科学基础知识库，结合教材内容说明扩散的影响因素

也可以使用简写口令：

- `使用材科基知识库`

为了得到更完整的回答，建议问题中补充：

- 你想要概念解释、章节定位，还是系统展开
- 是否需要结合教材结构回答
- 是否需要给出对应页码或出处线索

## 6. 适合的使用场景

- 教材复习
- 考试准备
- 课程答疑

## 7. 分享边界

本仓库适合：

- 个人使用
- 学习小组内部共享
- 小范围协作问答和检索

不建议：

- 公开传播
- 镜像到公开仓库
- 在未经授权情况下二次分发原始资料

## 8. 一句话使用建议

想快速看结论，先看 `08_outputs/`。

想找章节位置，先看 `04_index/`。

想做更扎实的问答，结合 `04_index/`、`03_chunks/` 和 `02_ocr_pages/` 一起使用。
