# 跨章节主线

## 强化机制总线

- 章节链路：第3章《晶体的塑性形变》 → 第4章《晶体中的缺陷》 → 第10章《回复与再结晶》 → 第11章《固态相变（一）扩散型相变》 → 第12章《固态相变（二）马氏体相变》
- 使用理由：把加工硬化、位错强化、再结晶软化、沉淀强化和马氏体强化串成一条主线。

## 热力学到相图到相变

- 章节链路：第5章《材料热力学》 → 第6章《相图》 → 第11章《固态相变（一）扩散型相变》 → 第12章《固态相变（二）马氏体相变》
- 使用理由：适合回答从自由能判据一路追问到组织演化和相变类型的问题。

## 缺陷到界面到扩散

- 章节链路：第4章《晶体中的缺陷》 → 第7章《界面》 → 第8章《固体中的扩散》
- 使用理由：适合解释原子迁移、界面迁移、晶粒长大和缺陷辅助扩散。

## 凝固到固态组织演化

- 章节链路：第6章《相图》 → 第9章《凝固与结晶》 → 第10章《回复与再结晶》 → 第11章《固态相变（一）扩散型相变》 → 第12章《固态相变（二）马氏体相变》
- 使用理由：适合做全流程组织演化分析，从铸态到热处理态。

## 结构基础到性能表现

- 章节链路：第1章《晶体学基础》 → 第2章《固体材料的结构》 → 第3章《晶体的塑性形变》 → 第4章《晶体中的缺陷》
- 使用理由：适合从晶体结构、键合、变形机制、缺陷结构一路解释性能来源。
