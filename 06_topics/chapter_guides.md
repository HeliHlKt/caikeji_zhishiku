# 章节导引

## 第1章 晶体学基础

- PDF页码：24-66
- 书内页码：1-43
- 本章定位：本章是全书的几何与结构语言基础，后续位错、相变、扩散与界面问题都要回到晶体学描述。
- 核心主题：空间点阵、晶胞与晶系、晶面晶向指数、常见晶体结构、倒易点阵
- 常见问法：如何标定晶面与晶向；FCC/BCC/HCP结构差异；倒易点阵有什么用
- 作答镜头：先定义对象 → 再给几何关系 → 接着说明结构特征 → 最后联系材料问题
- 考试抓手：晶面指数与晶向指数计算；密排面与密排方向判断；晶体结构几何特征比较

## 第2章 固体材料的结构

- PDF页码：67-170
- 书内页码：44-147
- 本章定位：本章把原子结构、键合方式与相结构联系起来，是理解结构稳定性、固溶强化与化合物形成的基础。
- 核心主题：原子结构、结合键、晶体电子结构、固溶体、离子化合物、金属间化合物
- 常见问法：结合键如何决定性能；Hume-Rothery规则怎么用；不同相结构为何稳定
- 作答镜头：先讲键合与电子结构 → 再讲结构稳定性 → 最后落到材料性能和相类型
- 考试抓手：离子键/共价键/金属键比较；固溶体形成条件；典型金属间化合物结构特点

## 第3章 晶体的塑性形变

- PDF页码：171-225
- 书内页码：148-202
- 本章定位：本章聚焦晶体塑性变形的微观机制，既解释材料为什么能变形，也解释为什么会强化、产生织构或走向断裂。
- 核心主题：滑移、Schmid定律、孪生、应变硬化、织构、断裂
- 常见问法：滑移与孪生怎么比较；单晶与多晶塑性有何不同；加工硬化为什么会发生
- 作答镜头：先给变形机制 → 再给晶体学条件 → 再讲组织演化 → 最后讲性能后果
- 考试抓手：Schmid定律应用；滑移与孪生比较；应变硬化机理；织构形成与影响

## 第4章 晶体中的缺陷

- PDF页码：226-338
- 书内页码：203-315
- 本章定位：本章是材料强韧化与塑性理论的核心章节，位错概念几乎贯穿后续回复再结晶、沉淀强化和马氏体讨论。
- 核心主题：点缺陷、位错、柏氏矢量、位错应力场、位错交互作用、晶界
- 常见问法：点缺陷与位错如何区分；位错为何控制强度和塑性；位错模型如何比较
- 作答镜头：先分缺陷类型 → 再讲几何和能量 → 再讲运动与交互作用 → 最后联系强化
- 考试抓手：位错定义与柏氏矢量；位错应力场与弹性能；位错增殖和塞积；晶界的位错观点

## 第5章 材料热力学

- PDF页码：339-370
- 书内页码：316-347
- 本章定位：本章提供全书的稳定性与驱动力语言，是分析相图、扩散、凝固和固态相变时最重要的判据基础。
- 核心主题：热力学第一定律、热力学第二定律、化学位、吉布斯自由能、相平衡、相变热力学

- 作答镜头：先写热力学函数 → 再给平衡判据 → 再落到自由能曲线 → 最后连接相图和相变
- 考试抓手：化学位意义；公切线法则；自由能曲线与相稳定性；脱溶与形核驱动力

## 第6章 相图

## 第7章 界面
